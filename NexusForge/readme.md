Este projeto faz parte do meu programa de desenvolvimento profissional dentro da Nexus Forge Studios, uma empresa fictícia de tecnologia e games criada para simular um ambiente real de mercado.

Nessa jornada, atuo como Desenvolvedor Web Estagiário, recebendo demandas que aumentam gradualmente em complexidade, desde projetos básicos até aplicações mais completas e profissionais.

O objetivo é desenvolver habilidades técnicas, boas práticas de programação e experiência próxima à realidade de uma equipe de desenvolvimento, criando projetos que também podem ser utilizados em meu portfólio.

Objetivos
    Praticar HTML, CSS e JavaScript.
    Desenvolver interfaces responsivas.
    Aprender organização e estruturação de projetos.
    Simular demandas reais do mercado de tecnologia.
    Construir um portfólio consistente.
    Evoluir progressivamente para projetos mais avançados.
